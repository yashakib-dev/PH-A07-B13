# KeenKeeper, Keep Your Friendships Alive

## Description
KeenKeeper is a responsive web application that helps users track and manage their friendships. Users can log interactions like calls, texts, and video chats, view them in a timeline, and analyze their communication patterns through a simple analytics dashboard.



## Technologies Used
- Next.js
- Tailwind CSS
- React Context API
- Recharts
- React Icons
- React Toastify
- DaisyUI


## Features
-  Interactions (Call, Text, Video Call)
-  View interaction history in a timeline
-  Filter timeline by interaction type
-  Visualize interaction data using a pie chart
-  Toast notifications for user actions
-  Fully responsive design (mobile, tablet, desktop)
